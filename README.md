# Agrinho - Agro Sustentável

Projeto de site sobre agro sustentável desenvolvido com HTML, CSS e JavaScript. O objetivo é apresentar práticas e informações relacionadas à agricultura sustentável de forma simples, acessível e educativa. O site foi construído com foco em responsividade, organização do conteúdo e experiência do usuário, utilizando tecnologias fundamentais do desenvolvimento web front-end.
